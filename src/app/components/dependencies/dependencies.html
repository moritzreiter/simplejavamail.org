Project dependencies

Simple Java Mail can be used without any external libraries, but a few features require additional dependencies.

CLI support

To enable CLI support, include the following module: 

<dependency>
    <groupId>org.simplejavamail</groupId>
    <artifactId>cli-module</artifactId>
</dependency>

DKIM support

To enable support for DKIM signing (see Features), include the following library: 

<dependency>
    <groupId>org.simplejavamail</groupId>
    <artifactId>dkim-module</artifactId>
</dependency>

Outlook support

To enable support for parsing Outlook messages (see Features), include the following library: 

<dependency>
    <groupId>org.simplejavamail</groupId>
    <artifactId>outlook-module</artifactId>
</dependency>

Spring support

To enable support for Spring (see Configuration), make sure the following libraries can be found on the classpath (in most cases will be there anyway): 

<dependency>
    <groupId>org.simplejavamail</groupId>
    <artifactId>spring-module</artifactId>
</dependency>
